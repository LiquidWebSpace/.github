# 👋 Welcome to LiquidWebSpace

**LiquidWebSpace** is a UK-based technology company building modern hosting, telecoms, automation and AI-powered software solutions.

We create and manage digital products that help businesses, developers and service providers work faster, support customers better, and automate everyday technical tasks.

---

## 🚀 What We Build

### 🌐 LiquidWebSpace

LiquidWebSpace provides web hosting, development, infrastructure and digital services.

We focus on reliable, practical and easy-to-manage solutions for websites, domains, hosting platforms, customer portals and business systems.

---

### 📞 LiquidVoice

**LiquidVoice** is our AI-powered voice and IVR platform.

It is designed to help businesses handle phone calls more intelligently using AI call routing, smart assistants, customer support automation and call handling workflows.

LiquidVoice helps businesses:

- Answer calls automatically
- Route customers to the correct department
- Handle common support queries
- Reduce missed calls
- Improve customer experience
- Connect voice calls into web portals and APIs

---

### 🤖 LiquidAI

**LiquidAI** is our AI workspace for web, desktop and mobile.

It is being built as a modern AI assistant platform for coding, project work, automation and connected workflows.

LiquidAI is planned for:

- Windows
- macOS
- Web
- iOS
- Android

The goal is to create an AI-powered workspace that can help users work across projects, files, APIs, chats and automation tools in one place.

---

### 🛠️ MyWebSitePanel

**MyWebSitePanel** is our hosting and website control panel brand.

It is designed to make website, hosting and account management easier for customers and internal teams.

MyWebSitePanel focuses on:

- Hosting management
- Domain tools
- Customer portals
- Website services
- Support workflows
- Account and billing integrations
